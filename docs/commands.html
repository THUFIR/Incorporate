General Company Commands

All commands have the structure /inc <action> [arguments]

list

Lists all the companies currently in the game.
Usage: /inc list

create

Creates a new company.
Usage: /inc create <company> <console: player>

delete

Delete a company.
Usage: /inc delete <company>
Permission: DELETE

rename

Rename a company.
Usage: /inc rename <company> <newName>
Permission: RENAME

Rank Commands

listRanks

List the ranks in a company.
Usage: /inc listRanks <company>
Alias: lr

addRank

Adds a rank to a company.
Usage: /inc addRank <company> <rank> <wage>
Alias: ar
Permission: MANAGERANKS

removeRank

Removes an existing rank from a company.
Usage: /inc removeRank <company> <rank>
Alias: rr
Permission: MANAGERANKS

getWage

Gets teh wage of a rank
Usage: /inc getWage <comapny> <rank>
Alias: gw

setWage

Sets the wage of a rank
Usage: /inc setWage <company> <rank> <wage>
Alias: sw
Permission: MANAGERANKS

getDRank

Gets the default rank of a company. The default rank in the company is the rank of new employees and of employees whose rank has been removed.
Usage: /inc getDRank <company>

setDRank

Sets the default rank of a company. Rank must exist.
Usage: /inc setDRank <company> <rank>
Permission: MANAGERANKS

Permissions
